GNOME Shell Lock Screen

GNOME Shell のロック画面で表示される時刻のフォントの上部が欠けていたので、暫定対策しました。

ロック画面のフォントの変更

font-family: "DejaVu Sans";でフォントファミリーを指定しています。

設定の反映

設定を反映するには、Alt + F2 を押下、r を入力してEnterで、GNOME Shell を再起動します。
