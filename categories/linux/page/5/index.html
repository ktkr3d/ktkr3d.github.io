2012-02-06
Ubuntu 11.10 + blender 2.61

https://launchpad.net/~irie/+archive/blender

Ubuntu 11.10 x86_64 にblender 2.61 を入れてみた。
blender 2.6 での日本語化は、

  1. 「File」-「User Preferences」メニューで「System」タブを選択

  2. 「International Fonts」をチェック

  3. 「Language」で「Japanese(日本語)」を選択

  4. 「Interface」と「Tooltip」をチェック

  5. 「デフォルトとして保存」を押す

Fennec を使ってみた

http://nightly.mozilla.org/

Fennec(モバイル/タブレット版のFirefox)をUbuntu 11.10 のGNOME Shell 上で使ってみた。
インタフェースはAndroid版のFirefox と似た感じで、同じようにタッチスクリーンの画面上を指でドラッグすることでスクロールもできた。左側の「+」ボタンでタブを追加。アドレスバーをタップするとページ一覧、ブックマーク、履歴、同期しているPCのタブ一覧を表示。今回使ったリビジョンでは残念ながら同期はうまく動かなかった。
