構成

機器 構成
VPN サーバ - Raspberry Pi 2
- Ubuntu 16.04 (ARM)
- OpenVPN 2.3.10
Linux クライアント - Ubutu GNOME 16.10 (x86_64)
- GNOME Shell 3.22.1
Android クライアント - Android 4.4.2
- OpenVPN アプリ
