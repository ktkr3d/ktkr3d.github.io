Tilix

Created2018-12-14|Updated2018-12-14

https://gnunn1.github.io/tilix-web/

ターミナルエミュレータTilix を使ってみました。

  • Ubuntu 18.10 x86_64
  • Tilix 1.8.1

Quake スタイルのドロップダウンターミナルとして、GNOME Shell 拡張のDropDownTerminal / DropDownTerminalX、Guake、Terminator 等を使ってきました。それらと比べて、Tilix はターミナルのタブ化の他、画面分割が柔軟にできて、とても便利です。

インストール

Tilix の起動設定

設定(gnome-control-center)の デバイス - キーボード - キーボードショートカット で、tilix --quake をF12 に割り当てます。

設定

  1. Tilix のタイトルバーのアイコンをクリックし、設定メニューを選択
  2. 外観 - Window Style -> Disable CSD
  3. 外観 - Theme variant -> 暗い
  4. Quake - Hide the tool bar of the window -> チェック
  5. ショートカット - アプリケーション - 設定を開く -> Alt+Return
  6. プロファイル - デフォルト - color - color schema -> Monokai Dark
  7. プロファイル - デフォルト - color - オプション - Transparancy -> 10% くらいに

使い方

  • Tilix の表示/非表示の切り替えはF12
  • 新規タブは右クリックのアイコン
  • タブの最大化/リストアは右上のアイコン
  • タブ間の切り替えはCtrl+Tab
  • 設定画面はAlt+Enter
