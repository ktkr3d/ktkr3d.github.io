  • nodejs 11.4.0

  • Visual Studio Code 1.30.0
    • Vue 2 Snippets
    • Vue VS Code Extension Pack
  • Google Chrome 71.0.3578.80
    • Vue.js devtools

サンプルVue.js アプリケーションの作成
