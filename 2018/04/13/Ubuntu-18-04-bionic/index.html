Ubutu 18.04 bionic beta2

DELL Inspiron 11 にUbuntu 18.04 bionic beta2 をインストールしてみました。

Ubuntu Livepatch も設定してみました。

インストールメディア

http://cdimage.ubuntu.com/daily-live/current/

デイリービルドのライブISOをEtcher でUSBメモリに格納しました。

GNOME の設定

  • ホームディレクトリの各フォルダ名を英語名に変更
  • テキストテンプレートファイルの作成
  • alias の設定
  • 基本的なパッケージのインストール
  • デスクトップ共有(Vino)の設定
    gnome-control-center で共有-画面共有をアクティブにする。
  • gnome-tweak-tool 関連
    • トップバー アプリケーションメニュー をオフ
    • トップバー 時計 日時 をオン
    • スタートアップアプリケーションにGuake Terminalを追加

GNOME Shell の設定

  • ブラウザ拡張のインストール
    https://extensions.gnome.org/
    Click here to install browser extension. をクリックしてインストール
  • chrome-gnome-shell パッケージのインストール
  • Extension のインストール
    • Dash to Panel
    • Argos

デスクトップのテーマの変更

  • Ubuntu コミュニティテーマのインストール gnome-tweak-tool でアプリケーションにCommunithemeを、アイコンにSuruを選択する。

アプリケーションのインストール

  • Google Chrome
    https://www.google.co.jp/chrome/index.html seahorseでパスワードをクリアしておく
  • Kodi

    Ubuntu Livepatch の設定

    software-properties-gtkのアップデートタブで、Ubuntu One にサインインし、Use Canonical Livepatch to increase security between restartsにチェックを入れる。
    6年前に作ったきりのアカウントでも問題なくログインできました。

問題

  • Ubuntu インストーラの次へ進むボタンが画面外に隠れてしまう。Enterキー押下で進んだ。
  • タッチパッドの右ボタンが左ボタンと同じ動作をする。マウスの右ボタンは正常に動作する。デフォルトの設定が変更されたためのようです。gnome-tweak-tool で以下の設定で解決しました。
