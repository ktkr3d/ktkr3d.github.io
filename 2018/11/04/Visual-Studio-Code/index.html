Visual Studio Code

Created2018-11-04|Updated2018-11-04

Atom よりVisual Studio Code の方が使われているのはどうしてだろう？
ということで、Visual Studio Code を使ってみます。

あらかじめ必要な機能が含まれている点や、性能が受け入れられている理由なんでしょうかね。

拡張機能

Activitus Bar

アクティビティバーのボタンをステータスバーに表示

Japanese Language Pack

日本語GUI

Project Manager

プロジェクト管理

Shortcuts

ステータスバーにショートカットを作成

良いところ

  • 軽快
  • カスタマイズしようと思っていたキーマップがデフォルトで設定されていた
  • git の基本操作がGUI から簡単にできる
  • 拡張機能の管理がすばやくできる
  • Project Manager がある
  • ターミナル機能が標準
