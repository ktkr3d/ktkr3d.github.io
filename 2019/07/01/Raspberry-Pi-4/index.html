Raspberry Pi 4

Created2019-07-01|Updated2019-07-01

Raspberry Pi 4 が発売されたので移行を検討してみます。

Raspberry Pi 2 Model B (2015/2/1 発売)からRaspberry Pi 4 Model B (2019/6/24 発売)への移行になりますので、4年間の大きな機能向上、性能向上が期待できそうです。反面、消費電力が大きくなってそうですが、Wi-Fi やBluetooth をオフにしておけば抑えることができるのかな。

ともかく、やっと32bit から抜け出せそうです！

スペック比較

発売日 2015年2月1日 2019年6月24日

900 MHz 1.5GHz

電源ソース USB micro-B USB Type-C

気になること

  • Ubuntu Server 18.04 LTS 64bit ARM イメージの公開時期
    https://wiki.ubuntu.com/ARM/RaspberryPi
  • 日本における技術基準適合証明の時期
    raspbian は国を指定するまでWi-Fiは有効にならないみたいですね
  • 消費電力
    どこまで低消費電力に設定できるかな
  • 積層ケース
    2F をRPi2に、4F をRPi4 に
