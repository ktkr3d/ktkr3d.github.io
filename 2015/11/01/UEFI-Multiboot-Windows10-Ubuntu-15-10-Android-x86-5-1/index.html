UEFI Multiboot Windows10, Ubuntu 15.10, Android-x86 5.1

Created2015-11-01|Updated2015-11-01|os

Inspiron 11 3158 (skylake Core i3-6100U, SSD 128GB)гӮ’е…ҘжүӢгҒ—гҒҹгҒ®гҒ§гҖҒгғһгғ«гғҒгғ–гғјгғҲз’°еўғгӮ’ж§ӢзҜүгҒ—гҒҰгҒҝгҒҫгҒ—гҒҹгҖӮ

Fast Boot гӮ’жҠ‘жӯў

  1. Windows гғңгӮҝгғігӮ’еҸігӮҜгғӘгғғгӮҜгҒ—гҒҰгҖҒ[йӣ»жәҗгӮӘгғ—гӮ·гғ§гғі]жҠјдёӢ
  2. [йӣ»жәҗгғңгӮҝгғігҒ®еӢ•дҪңгӮ’йҒёжҠһгҒҷгӮӢ]жҠјдёӢ
  3. [зҸҫеңЁеҲ©з”ЁеҸҜиғҪгҒ§гҒҜгҒӘгҒ„иЁӯе®ҡгӮ’еӨүжӣҙгҒ—гҒҫгҒҷ]жҠјдёӢ
  4. [й«ҳйҖҹгӮ№гӮҝгғјгғҲгӮўгғғгғ—гӮ’жңүеҠ№гҒ«гҒҷгӮӢ(жҺЁеҘЁ)]гҒ®гғҒгӮ§гғғгӮҜгӮ’еӨ–гҒ—гҖҒ[еӨүжӣҙгҒ®дҝқеӯҳ]жҠјдёӢ
  5. Windows 10 гӮ’гӮ·гғЈгғғгғҲгғҖгӮҰгғігҒ—гҒҫгҒҷгҖӮ

Ubuntu гҒ®гӮӨгғігӮ№гғҲгғјгғ«

  1. д»ҘдёӢгҒ®64bit зүҲUbuntu гҒ®ISO гӮ’Unetbootin гҒ§USB гғЎгғўгғӘгҒ«жӣёгҒҚиҫјгҒҝгҖҒUSB гӮ№гғӯгғғгғҲгҒ«жҢҝе…ҘгҒ—гҒҫгҒҷгҖӮ
    http://cdimage.ubuntu.com/ubuntu-gnome/releases/15.10/release/ubuntu-gnome-15.10-desktop-amd64.iso

  2. USB гҒӢгӮүUbuntu гӮ’гғ©гӮӨгғ–гғўгғјгғүгҒ§иө·еӢ•гҒ—гҒҫгҒҷгҖӮ
    PC гҒ®гғңгғӘгғҘгғјгғ гғһгӮӨгғҠгӮ№гғңгӮҝгғігӮ’жҠјгҒ—гҒӘгҒҢгӮүгҖҒйӣ»жәҗгғңгӮҝгғігӮ’жҠјдёӢгҒ—гҖҒгғӯгӮҙгҒҢеҮәгҒҹгӮҝгӮӨгғҹгғігӮ°гҒ§гғһгӮӨгғҠгӮ№гғңгӮҝгғігӮ’йӣўгҒ—гҒҫгҒҷгҖӮ

  3. gparted гҒ§гғ‘гғјгғҶгӮЈгӮ·гғ§гғігҒ®гғ¬гӮӨгӮўгӮҰгғҲгӮ’еӨүжӣҙгҒ—гҒҫгҒҷгҖӮ
    Ubuntu гҒ®гӮӨгғігӮ№гғҲгғјгғ«з”ЁгҒ«35GB зЁӢеәҰгӮ’зўәдҝқгҒҷгӮӢгҒҹгӮҒгҒ«гҖҒWindows гҒ®й ҳеҹҹгӮ’зё®е°ҸгҒ—гҒҰгҖҒе…ЁдҪ“гӮ’еүҚж–№гҒ«з§»еӢ•гҒ—гҒҫгҒҷгҖӮ

  4. Ubuntu гҒ®гӮӨгғігӮ№гғҲгғјгғ«гҒ§гҒҜгҖҒ2гҒӨгҒ®гғ‘гғјгғҶгӮЈгӮ·гғ§гғігӮ’дҪңжҲҗгҒ—гҒҫгҒҷгҖӮ

    • гғ«гғјгғҲгғ‘гғјгғҶгӮЈгӮ·гғ§гғі(/dev/sda6, ext4, 30GB)
    • swap гғ‘гғјгғҶгӮЈгӮ·гғ§гғі(/dev/sda7, swap, 5GB)

  5. гғ–гғјгғҲгғӯгғјгғҖгҒҜгҖҒ/dev/sda гӮ’жҢҮе®ҡгҒ—гҒҫгҒҷгҖӮ
    /dev/sda й…ҚдёӢгҒ®EFI гғ‘гғјгғҶгӮЈгӮ·гғ§гғі(/dev/sda1)гҒ«гғ–гғјгғҲгғӯгғјгғҖгҒҢгӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгҒҫгҒҷгҖӮ

  6. еҶҚиө·еӢ•гҒ—гҒҫгҒҷгҖӮGRUB гҒ«д»ҘдёӢгҒҢиЎЁзӨәгҒ•гӮҢгӮӢгӮҲгҒҶгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

Secureboot гҒ®з„ЎеҠ№еҢ–

  1. PC гӮ’иө·еӢ•гҒ—гҖҒGRUB дёҠгҒ®System Setup гӮ’йҒёжҠһгҒ—гҒҫгҒҷгҖӮ
  2. Secureboot гӮ’з„ЎеҠ№гҒ«гҒ—гҒҫгҒҷгҖӮ

Android-x86 гҒ®гӮӨгғігӮ№гғҲгғјгғ«

  1. д»ҘдёӢгҒ®64bit зүҲAndroid-x86 гҒ®гӮӨгғЎгғјгӮёгӮ’dd гҒ§USB гғЎгғўгғӘгҒ«жӣёгҒҚиҫјгҒҝгҖҒгӮ№гғӯгғғгғҲгҒ«жҢҝе…ҘгҒ—гҒҫгҒҷгҖӮ
    http://sourceforge.net/projects/android-x86/files/Release%205.1/android-x86_64-5.1-rc1.img/download
  2. PC гҒ®гғңгғӘгғҘгғјгғ гғһгӮӨгғҠгӮ№гғңгӮҝгғігӮ’жҠјгҒ—гҒӘгҒҢгӮүгҖҒйӣ»жәҗгғңгӮҝгғігӮ’жҠјдёӢгҒ—гҖҒгғӯгӮҙгҒҢеҮәгҒҹгӮҝгӮӨгғҹгғігӮ°гҒ§гғһгӮӨгғҠгӮ№гғңгӮҝгғігӮ’йӣўгҒ—гҒҫгҒҷгҖӮ
  3. Android-x86 гҒ®гӮӨгғігӮ№гғҲгғјгғ©гҒҢиө·еӢ•гҒ•гӮҢгӮӢгҒ®гҒ§гҖҒ/dev/sda6 гҒ«гӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҫгҒҷгҖӮгғ–гғјгғҲгғӯгғјгғҖгҒҜгӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҫгҒӣгӮ“гҖӮ
  4. еҶҚиө·еӢ•гҒ—гҒҫгҒҷгҖӮ

Secureboot гҒ®жңүеҠ№еҢ–

  1. PC гӮ’иө·еӢ•гҒ—гҖҒGRUB дёҠгҒ®System Setup гӮ’йҒёжҠһгҒ—гҒҫгҒҷгҖӮ
  2. Secureboot гӮ’жңүеҠ№гҒ«гҒ—гҒҫгҒҷгҖӮ

Android-x86 гҒ®иө·еӢ•гғЎгғӢгғҘгғјиҝҪеҠ 

  1. Ubuntu гҒ§иө·еӢ•гҒ—гҖҒд»ҘдёӢгҒ®гғ•гӮЎгӮӨгғ«гӮ’з·ЁйӣҶгҒ—гҒҫгҒҷгҖӮ
  2. иЁӯе®ҡгӮ’еҸҚжҳ гҒ—гҒҫгҒҷгҖӮ
