コマンド

コマンド 別名 説明
hexo new n 新しい記事の作成
hexo generate g 静的ファイルの生成
hexo server s ローカルサーバの起動
hexo deploy d Web サイトへデプロイ
hexo new page 新しいページの作成
hexo clean 生成したファイルとキャッシュの消去

404 ページ

404.md というファイルを作成しておくとGitHub のサイト上に存在しないパスが指定された場合に表示されるページを生成できます。
