Unity 3D Android Apps

https://www.assetstore.unity3d.com/jp/?gclid=CMiM1LCS0McCFcSUvQodUEYLYQ#!/content/40756
Unity 3D のサンプルをAndroid 向けにビルドしてみました。

環境は

  • Ubuntu 15.10 (wily) x86_64 Beta
  • GNOME Shell 3.17.90
  • Linux kernel 4.1.0
  • Android 4.4.2

Andorid SDK

Android SDK のLinux 版をダウンロードして展開します。

http://developer.android.com/sdk/index.html#Other

JDK

JDK のLinux 版をダウンロードして展開します。

http://www.oracle.com/technetwork/java/javase/downloads/index.html

ライブラリの追加

ビルド中にFailed to re-package resources. See the Console for details. というエラーを抑止するために以下を実行しておきます。

Android SDK, JDK の位置の指定

  1. Unity Editor で、[Edit]-[Preferences] でUnity Preferences 画面を表示
  2. External Tools のAndroid SDK Location とJDK Location を設定

サンプルプロジェクトのインポート

  1. [Window]-[Asset Store] を開いて、サンプルプロジェクトをダウンロード、インポート
  2. Project タブ内のAssets から.unity ファイル(Scenes 下とかにあるみたい)をダブルクリック

Android アプリのビルド

  1. [File]-[Build Settings…]を選択
  2. Platform でAndroid を選択
  3. Player Settings… ボタンを押下
  4. Unity 側のOther Settings タブのIdentification のBundle Identifier の値を変更
  5. Build Settings 画面に戻ってBuild ボタンを押下
  6. 出力するファイル(.apk)を指定
