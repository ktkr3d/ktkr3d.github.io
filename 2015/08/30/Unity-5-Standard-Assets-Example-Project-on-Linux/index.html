Unity 5 Standard Assets Example Project on Linux


http://forum.unity3d.com/threads/missing-default-assets.350369/

Linux 版のUnity のインストーラでは自動でStandard Assets がインストールされないので手動でインストールしてみました。

Standard Assets の取得

  1. [Window]-[Asset Store] を開く
  2. Standard Assets をダウンロードする
    https://www.assetstore.unity3d.com/jp/?gclid=CMWs08L50McCFRQDvAodM6QPbg#!/content/32351

Standard Assets Example Project の作成

  1. New Project でプロジェクトを新規作成する。
  2. Project name にStandard Assets Example Project を指定する。
  3. Asset packages ボタンを押下して、Standard Assets をチェックする。
  4. Create Project ボタンを押下する。
  5. [File]-[Save Project] を選択する。

サンプルの実行

  1. Assets/Sample Scenes/Scenes/ を開く。
  2. 例えば、Car.unity をダブルクリックする。
  3. [Edit]-[Play] でサンプルを実行する。
