MMD for Unity on Linux

http://mmd-for-unity-proj.github.io/mmd-for-unity/

Unity 3D のLinux 版でMMD を取り込んでみました。

MMD for Unity のダウンロード

  1. http://mmd-for-unity-proj.github.io/mmd-for-unity/ を開きます。
    DOWNLOADS ボタンのリンク先のマスターブランチは少し古いのでブランチを切り替えることにします。
  2. View on GitHub ボタンをクリックして、リポジトリでブランチを2.1.1b に切り替えます。
  3. Download ZIP をクリックしてダウンロードして、プロジェクトのAssets ディレクトリに展開します。
  4. メニューに[MMD for Unity] が表示されます。

PMD のインポート

  1. Assets の中のPMD モデルファイルを選択します。
  2. Inspector でConvert to Perfabを選択します。

とやったような気がする。
