Flutter

Flutter の環境を構築してサンプルアプリケーションを実行してみました。

環境

  • Windows 10
    • flutter 2.0.1
    • git 2.30.0
    • android studio 4.1.2

構築

Flutter のインストール

  1. Download
    https://flutter.dev/docs/get-started/install/windows
    flutter_windows_2.0.1-stable.zip
  2. 展開
    C:\tools\flutter
  3. 環境変数PATH の追加
    C:\tools\flutter\bin

Android Studio のインストール

  1. Andoird Studio インストール
    https://developer.android.com/studio
    Web アプリのみの場合は不要かもしれません。
  2. Configure > AVD Manager
    AVD の起動を確認して終了
  3. Configure > Plugins
  • Dart
  • Flutter

VSCode プラグインのインストール

  1. VSCode プラグインのインストール
  • Dart
  • Flutter
  • flutter Widget Snippets

診断

  1. Android ライセンス使用許諾の設定 ‘y’ を押下して許諾
  2. flutter doctor

実行

  1. F1 > flutter
    Flutter: New Application Project
  2. プロジェクトフォルダを選択
    C:\repos\flutter_sample
  3. 実行
    VSCode の右下からデバイスを選択して実行
