Windows Terminal

Created2021-03-08|Updated2021-03-08

久しぶりにWindows Terminal を使ってみました。
背景透過の設定、ドロップダウンターミナル的な使い方の設定をしてみました。

環境

  • Windows 10
    • Windows Terminal Preview バージョン: 1.7.572.0

構築

  1. ダウンロード
    https://github.com/microsoft/terminal/releases
  2. インストールして実行
  3. Windows タスクバーの設定
  • 右クリックして「タスクバーにピン留めする」を選択
  • アイコンをタスクバー の一番左に移動(Windows キーと1キーで起動するため)
  1. 起動サイズの指定
  2. 背景色透過

実行

  1. Windows + 1 で起動と最小化/元に戻すを切り替え

利用

操作 ショートカットキー
画面分割 Alt + Shift + +キー
画面分割 Alt + 上部の+アイコン
ペイン移動 Alt + 矢印キー
表示の切り替え Windows + 1
