EPGStation/Mirakurun on WSL

Created2020-07-12|Updated2020-07-12


https://github.com/l3tnun/EPGStation
https://github.com/Chinachu/Mirakurun
Chianchu の調子が悪くなったのでEPGStation に乗り換えてみました。
EPGStation とMirakurun をWSL 上に構築しました。

環境

  • ソフトウェア構成

    • Windows 10 Insider Preview build 20161
      • WSL (Ubuntu 20.04 LTS)
        • node v14.4.0
          • EPGStation 1.7.0
          • Mirakurun 3.2.0
        • npm 6.14.5
        • pm2 4.4.0
        • ffmpeg 4.2.2
        • python 2.7.18rc1
        • gcc 9.3.0
      • BonRecTest 1.0.0
      • Kodi 18.6
        • PVR IPTV Simple Client 3.9.8

  1. 非表示で実行するためのランチャの作成

  1. ポート開放
  • 8888 (EPGStation)
  • 40772 (Mirakurun) オプション

利用

  1. Mirakurun の動作確認
    http://192.168.11.2:40772
  2. EPGStation にアクセス
    http://192.168.11.2:8888
  3. ライブ視聴は「GR」→ 局名 →「視聴」を選択して、VLC やMX Player MX Player 等で開く

Kodi でライブ視聴

https://github.com/l3tnun/EPGStation/blob/master/doc/kodi.md

  1. アドオンをインストール
    「アドオン」→「ダウンロード」→「PVRクライアント」→「PVR IPTV Simple Client」
  2. インストールが完了したら「C」キーを押下して、コンテキストメニューから「設定」を選択
  3. 一般 / M3U プレイリストのURL
    http://host:port/api/iptv/channel.m3u8?mode=2
  4. EPG 設定 / M3U プレイリストのURL
    http://host:port/api/iptv/epg.xml?days=3
  5. Kodi を再起動
  6. 「TV」メニューから視聴
