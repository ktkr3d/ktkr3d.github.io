UbuntuDDE Remix 20.04 Beta


https://ubuntudde.com/
https://ubuntudde.com/blog/ubuntudde-beta-200410-release-note/

UbuntuDDE Remix 20.04 Beta を試してみました。
デザインが綺麗なディストリビューションです。

Windows 10 ホストのVirtualBox 上のVM として構築しました。
WSL2 とVirtualBox は共存するようになったんですね。

環境

  • Windows 10 Insider Preview Build 19613
    • VirtualBox 6.1.6
      • UbuntuDDE Remix 20.04 Beta
        • CPU x2
        • Memory 4GB
        • Disk 25GB

構築

インストール

  1. コントロールパネル」-「プログラムと機能」-「Windows の機能の有効化または無効化」で「仮想マシン プラットフォーム」を有効にする。
  2. VirtualBox で新規VM を作成し、UbuntuDDE のISO イメージから起動する。
  • Memory 4GB
  • Disk 25GB
  • EFI を有効化
  1. モードを選択する。
  • Effect Mode
  • Normal Mode
  1. インストーラを実行する。

コメント

  • デザインが綺麗
  • 外観の設定
  • ターミナルのQuake モード(タブ付)が標準で実装されている!!
