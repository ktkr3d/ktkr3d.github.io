Terminus

https://eugeny.github.io/terminus/
Windows の端末エミュレータをTerminus に切り替えました。

いろいろありますね。

  • ConEmu/Cmder
    https://conemu.github.io/
    https://cmder.net/
  • Fluent Terminal
    https://github.com/felixse/FluentTerminal
  • Terminus
    https://github.com/Eugeny/terminus
  • Windows Terminal
    https://github.com/microsoft/terminal

環境

  • Windows 10 Insider Preview 19628
    • Terminus 1.0.111

構築

インストール

  1. ダウンロード
    https://github.com/Eugeny/terminus/releases/latest
  2. インストール

設定のカスタマイズ

  • 自動起動
    • ファイル名を指定して実行でshell:startup
    • Terminus のショートカットを登録
    • 実行時の大きさを最小化に設定
      有効にならない場合があるかもしれない
  • Application 設定
    • Theme をCompact
    • Tabs location をAt the bottom
    • Dock the terminal をTop
      Windows タスクバーを右側に配置している場合はオフにして位置を調整する
    • Dock always on top をオフ
  • Hotkeys 設定
    • Toggle Terminal window にCtrl+F1 を追加
  • Plugins
    • save-output を有効化
  • SSH 設定
    • プロファイルを登録

利用

  • 表示/非表示(Ctrl + Space または Ctrl + F1)
  • 拡大(Ctrl + ^)、縮小(Ctrl + -)
  • 新しいタブ(Ctrl + Shift + T)
  • 右に分割(Ctrl＋Shift＋E)

良さそうなところ

  • ドロップダウン的に使えます
  • ConEmu よりも描画が安定しているように感じます
  • マルチプラットフォーム
  • プラグイン

これから確認するところ

  • Ctrl + Wheel で文字の拡大縮小をしたい
  • 日本語化
