Fractional Scaling on Wayland


CHUWI MiniBook X(10.51 inch, 1200*1920px)のように画素密度の高い画面ではスケーリング値を125%にすると見やすくなるとともにタッチスクリーン操作がしやすくなります。出荷時のWindows 11は125%になっています。これをLinux でもやってみましょう。

環境

  • Ubuntu 23.10 / GNOME Shell 45.1
  • Arch Linux / GNOME Shell 45.1
  • CHUWI MiniBook X N100
    画素密度： 215ppi
    画素ピッチ： 0.118mm
    総画素数： 2,304,000ピクセル
    スペック： 10.51インチ(1920×1200)
