Lutris


https://lutris.net/

Lutris でWorld of Warcraft を実行してみました。

  • OS はUbuntu 22.04.3 LTS, GNOME Shell 42.9, X11(X.org), Pulseaudio
  • wine-ge-8-24, DirectX 11モード, 64bit, dxvk, vkd3d

結果としては、

  • FPS が90超えることもあります。Windows 11(同じPC)と同等の性能です。
  • Shaderのコンパイルが発生すると左下にCompiling Shaders...と表示されます。若干もたつく感じがします。キャッシュ方法があるのかよくわかりません。
  • キーボードの反応が遅れることがあります。
  • 極まれに音飛びします。
  • DungeonやRaidには若干不安があるがファーミングには使えそう。

環境

  • Ubuntu 22.04.3 LTS, kernel 6.2.0-37
  • GNOME Shell 42.9, X11(X.org), Pulseaudio
  • Memory 32GiB
  • CPU i7-4790 4.00GHz x8
  • Geforce GTX 1060 6GB
  • Lutris
  • Runner はWine-ge-8-24
  • Game options - Argumentsに-dx11オプションを指定

構築

  • Lutris のインストール
  • Battle.net のインストールの場合
    1. 左上の+ボタンをクリック
    2. Searchメニューで Battlenetを検索して選択、インストールを実行
    3. 言語選択画面が表示されたら適当に選択(Englishで)
    4. ログイン画面は閉じる(重要)
    5. インストール完了を待つ
    6. Configureメニューを選択
    7. パラメタに-dx11を指定
    8. Battle.Net アプリのショートカットを作成
    9. Battle.Net アプリを起動してログイン
    10. ゲームをインストールしているwindowsのボリュームをマウント(ディスク アプリでWindows ボリュームを起動時に自動マウントできる)
    11. ゲームディレクトリを指定

実行

  1. Battle.Net アプリを起動
  2. World of Warcraft を起動
