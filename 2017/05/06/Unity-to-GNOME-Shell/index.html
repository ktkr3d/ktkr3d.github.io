Unity to GNOME Shell

Created2017-05-06|Updated2017-05-06

http://cdimage.ubuntu.com/ubuntu/
http://ktkr3d.github.io/linux/
Ubuntu 17.10 のデイリービルド版が公開されました。Ubuntu 17.10 からはデスクトップ環境がGNOME Shell に変更になる予定ですが、まだUnity の構成のようです。そこで、デスクトップ環境をUninty からGNOME Shell に変更してみました。

Unity から GNOME Shell へ変更

  • GNOME Shell のインストール
  • ウィンドウのボタン位置の変更(「右寄せ、最小化-最大化-閉じる」の場合)
  • Adwaita テーマのインストール
  • GNOME 壁紙のインストール

GNOME Shell 拡張の有効化

gnome-tweak-tool の拡張機能タブでAlternate Tab を有効にします。

GNOME Shell 拡張の追加インストール

  • Dash to Panel
  • Panel OSD
  • Topicons Plus
  • Workspaces to Dock

ディスプレイマネージャをGDM に変更する場合

  • LightDM からGDM へ切り替え
  • LightDM からGDM へ切り替え(インストール後に切り替える場合)
