Inspiron Duo でいろんな OS を動かして遊んでみよう。

  • Fedora 15 (DE: GNOME Shell)
  • Ubuntu 11.04 (DE: Unity)
  • MeeGo 1.2 Tablet UX Pre-Alpha Release
  • MeeGo v1.1 for Netbooks
  • MeeGo v1.1 for In-Vehicle Infotainment
  • Android-x86 2.2 / 2.3
  • Splashtop OS 1.0
  • Windows 7
  • Windows XP
