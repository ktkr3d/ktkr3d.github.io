Ubuntu 11.04 でGNOME Shell

http://www.ubuntuupdates.org/ppa/gnome_3?dist=natty

Ubuntu 11.04 用のGNOME Shell を含むppa を見つけたので、早速インストールしてみた。ちゃんとローカライズもされている。

GDM でログインする際に「Ubuntu GNOME Shell Desktop」を選択する。

上部パネルの日付、時刻の書式はスクリプトをカスタマイズ。シャットダウン系のメニューもスクリプトなので後で改造しよう。
