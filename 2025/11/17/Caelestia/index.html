Caelestia

Created2025-11-17|Updated2025-11-22

Hyprland用のCaelestia dotsを試してみました。これは良いかもしれない。

https://github.com/caelestia-dots/caelestia
https://github.com/caelestia-dots/shell

https://www.youtube.com/watch?v=TggHDm0_vBw

環境

ハードウェア

  • デスクトップPC x86_64、日本語キーボード配列、4Kディスプレイ
  • 2in1 PC N100、英語キーボード配列

ソフトウェア

  • Arch Linux
    • gdm 49.1
    • hyprland 0.25.1
    • quickshell 0.2.0
      • caelestia-shell 1.3.3
    • fish 4.2.1

インストール

  • バックアップ
    設定ファイルが上書きされるのでバックアップ推奨です。
    上書き確認はあります。

  • Caelestia dots のインストール

    インストールスクリプトはfishが前提です。

    qqc2-desktop-style-5.116.1.tar.xz ... 失敗 (不明な公開鍵 58D0EE648A48B3BB)
    というようなエラーがでた場合は以下のコマンドで更新して再実行します。
    gpg --keyserver keys.gnupg.net --recv-key 58D0EE648A48B3BB

  • Todoist のインストール

設定

  • ディスプレイの解像度を指定する場合
  • ディスプレイの回転(270°=90°*3)を指定する場合
  • 開いているウィンドウ数に関わらずウィンドウGapを統一する場合
  • 日本語キーボード配列の場合
  • 日本語入力を有効化する場合
  • よく使うアプリの指定
  • 壁紙変更
    1. ~/Pictures/Wallpapers/に画像を格納
    2. SUPER then >
    3. Wallpaperを選択
    4. 画像を選択
  • Caelestiaカスタム用設定ファイルの作成
    https://github.com/caelestia-dots/shell#user-content-example-configuration
    Exampleを展開表示して内容を~/.config/caelestia/shell.jsonに保存
  • ログアウト処理の是正（ログアウトを選択してもGDMに戻らず再起動してしまう場合）
    https://github.com/caelestia-dots/shell/issues/816#issuecomment-3483658302
  • N100で音声出力デバイスがダミー出力になってしまった場合
    PulseaudioからPipewireへ切り替え
  • ロック時間の設定

利用

キーバインド

分類 操作 動作
アプリケーション SUPER Open Launcher
  SUPER + T Execute Terminal
  SUPER + W Execute Web Browser
  SUPER + C Execute Code Editor
  SUPER + E Execute File Explorer
ウィンドウ SUPER + Q Close Active Window
  SUPER + ← / → / ↑ / ↓ Move Focus
  ALT + TAB Cycle Focus
  SUPER + Alt + 1 ( 〜 0 ) Move window to workspace #
  Ctrl + SUPER + Shift + ← / → Move Window to Workspcae
ワークスペース SUPER + 1 ( 〜 0 ) Switch Workspaces
  Ctrl + SUPER + ← / → Move to Workspcae
特別ワークスペース Ctrl + Shift + Esc Toggle System Monitor (btop)
  SUPER + M Toggle Music (Spotify)
  SUPER + D Toggle Communication (Discord)
  SUPER + R Toggle Todo (Todoist)
  SUPER + S Toggle Special Workspace
ユーティリティー SUPER then > List Utilities
  SUPER + V Clipboard Manger
  SUPER + . Emoji Selector
  SUPER + K Show Panels
  SUPER + Shift + M Mute
セッション Ctrl + Alt + Delete Open session menu
  SUPER + L Lock Screen
その他 Ctrl + Alt + C Clear Notifs
マウス/タッチパッド SUPER + Mouse Scroll Scroll Thru Workspaces
  SUPER + Left Mouse Click Drag Window
  SUPER + Right Mouse Click Resize Window

ユーティリティ (SUPER then >)

Name Description
Caluculator Do simple math equations (powered by Qalc)
Scheme Change the current colour scheme
Wallpaper Change the current wallpaper
Variant Change the current scheme variant
Random Switch to a random wallpaper
Light Change the scheme to light mode
Dark Change the scheme to dark mode
Lock Lock the current sessio
Sleep Suspend then hibernate

To Do

  • リモートログイン
    • GNOME Remote Desktopを利用する場合
      接続後、GDM画面でHyprlandセッションが選択肢に表示されない。
      gdmの設定を見直す必要があるかもしれない。
    • wayvncを利用する場合
      GNOME Remote Desktopを停止して動作するか確認してみる。
  • VPN
    shell.jsonにvpnセクションがあり、デフォルトは無効化状態になっている。
    wireguardをopenvpnに変更して有効化可能か確認する。
  • ランチャで日本語名のアプリをアルファベットで絞り込む方法
    「ファイル」アプリについてfile、nautilusのキーワードで絞り込めない。
  • 背景に時刻表示
    shell.jsonで有効に切り替え 画面右下に表示されるようになるけれど不要なので無効のままにする。
  • ~/.face
  • NumLockが外れる場合がある
  • keylayoutの再読み込みでUnknownになる
  • マルチモニタの確認
  • bluetoothの確認
  • starship
    テーマを元に戻したい
  • game mode
    Quick Toggles の右側から二番目
  • Installing VSCode/VSCodium configs
    https://github.com/caelestia-dots/caelestia?tab=readme-ov-file#installing-vscodevscodium-configs
  • メディアアプリの切り替え
    画面上部パネル > Media > 🔽からアプリを選択
  • Gapをさらに調整したい場合addreservedを使う
