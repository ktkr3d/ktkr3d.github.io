Prism Launcher

Created2025-12-21|Updated2025-12-22

Prism Launcherを使ってMinecraftをセットアップしました。過去にWindowsで遊んでいたワールドデータをCachyOS上で取り込めました。

Prism Launcher
An open source Minecraft launcher with the ability to manage multiple instances, accounts and mods. Focused on user freedom and free redistributability.

環境

  • CachyOS
    • Hyprland
    • Prism Launcher 9.4
      • Minecraft 1.21.8

構築

Prism Launcher

  • Prism Launcher
  • Minecraft
    1.21.8
    使いたいModsが対応している最新版を選択します。
  • Mods ローダ
    Fabric

日本語フォント

  • 日本語フォント

Mods

  1. マインクラフトのバージョン1.21.8を選んで編集。
  2. Modを選択してDownload mods
  3. 以下Mod(および前提Mod)を検索して導入
    • Fabric API
    • Iris
    • MiniHUD
      • MaliLib
    • Mod Menu
    • Modern UI
      • Forge Config API Port
    • RPG-Hud
    • Sodium
    • Xaero’s Minimap
    • Xaero’S World map

ワールドデータの追加

  • 過去に遊んだワールドを追加してみます。
    • コピー元(Windows)
    • コピー先(CachyOS)

アプリケーションメニュー化

  1. Prism Launcherでバージョン1.21.8を選んでショートカットを作成をクリックするとデスクトップに1.21.8.desktopが作成される。
  2. ファイル名をわかりやすいものに修正する。(例：Minecraft 1.21.8.desktop)
  3. アプリケーション名も同様に修正する。(例：Minecraft 1.21.8)
  4. ~/.local/share/applications/へ移動する。

起動

  1. 任意のランチャからMinecraft 1.21.8を起動
  2. ワールド「新規ワールド (15)」を選択

設定

Minecraft

  • 移動時の画面の揺れを固定にする
    「設定…」「ビデオ設定…」「一般」「画面の揺れ」をオフ
  • 音の字幕を表示する
    「設定…」「サウンド設定…」「字幕の表示」をオン

Modern UI

  • 日本語フォントの指定
    Modern UI で「Preferences」「Font」「preferred Font Family」「Choose a Font」でRounded M+ 1c regularを指定する。

MiniHUD

  • バイオーム名を表示
    MiniHUDでinfoBiomeを有効にする

フルスクリーンの設定

  • Hyprlandショートカットで都度フルスクリーンにする場合
    Super+F
  • Minecraftの設定で永続的にフルスクリーンにする場合
    「設定…」「ビデオ設定…」「一般」「フルスクリーン」をオン
