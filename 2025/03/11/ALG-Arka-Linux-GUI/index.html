ALG - Arka Linux GUI

Created2025-03-11|Updated2025-03-12

Arch LinuxのGUIインストーラであるALG(Arka Linux GUI)を試してみました。
https://github.com/arch-linux-gui/alg-releases

Welcomeアプリ(alg-welcome)とOSインストーラ(calamares)で構成されているようです。
Calam-Arch-Installerのようにインストール中にデスクトップ環境を選択することはできないようですが、インストール時やインストール後に日本語が表示されるのは便利ですね。

私はGNOMEを利用するのですが、alg-gnomeとalg-gnome-pureでは以下の違いがありました。

  • alg-gnome
    https://github.com/arch-linux-gui/alg-gnome/blob/master/packages.x86_64
    Firefox、LibreOffice、VLC、nvidiaなどのソフトウェアがインストールされる
    AURヘルパーのparuがインストールされる
    拡張機能(Apps Menu、User Theme)が有効になっている
    アイコン(tela-circle-icon)や壁紙(archlinux-wallpaper)がカスタマイズされている
    インストール時、インストール後の日本語表示(noto-fonts-cjk)に対応しているが日本語入力はできないっぽい
  • alg-gnome-pure
    https://github.com/arch-linux-gui/alg-gnome-pure/blob/main/packages.x86_64
    GNOME標準のソフトウェアがインストールされる
    日本語フォントは自分でインストールする必要がある

利用 (alg-gnome-pureの場合)

  1. Welcomeアプリ

    Update Mirrorlist (Japanを選択)

    Update System
  2. 日本語フォントのインストール (インストーラの日本語表示用)
    スーパーキー + Aを押下してアプリケーション一覧からTerminalを起動

/usr/share/fonts/OTF/*.ttfにインストールされる
3. Arch Linuxのインストール
WelcomeアプリでInstall ALGを選択
自動ログインにしておくと便利
4. 日本語フォントのコピー (初回ログイン後の日本語表示用)


5. シャットダウン
6. インストールメディアの取り外し
7. 起動
8. Welcomeアプリ
Update Mirrorlist (Japanを選択)
Update System
9. 鍵の更新 (時間がかかる)

10. アップデート
